# 📚 KnowledgeHUB – BookSphere  
### Smart Library Management System (Spring Boot + Angular)

A full-stack Library Management System built using Spring Boot and Angular with features like authentication, book tracking, request management, and admin workflows.

---

## 🚀 Status
**Phase 0 – Day 1 Completed**

- Spring Boot project initialized  
- Core dependencies added (Web, JPA, MySQL, Lombok, Validation)  
- Application running successfully  
- Basic `/hello` API implemented and tested  

## 🛠 Tech Stack
- Java  
- Spring Boot  
- Spring Data JPA  
- MySQL  
- Lombok  

## ▶️ Run Locally
```bash
git clone <repo-url>
cd knowledgehub-booksphere
./mvnw spring-boot:run
```
